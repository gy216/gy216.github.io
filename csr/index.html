Cloudflare 如何计算碳减排

对比计算法

Cloudflare 将使用其他 Internet 服务完成的数据传输量所产生的平均碳排放量，与使用 Cloudflare 服务的估计碳排放量进行对比，由此计算您节省的碳排放量。

可再生能源支持

除了帮助建立更好的 Internet 这一使命外，Cloudflare 还热衷于保护我们生活的这个世界。选择使用 Cloudflare 不仅可以减少碳排放总量，Cloudflare 还会代表所有客户购买可再生能源额度 (REC)，100% 通过可再生能源来支持 Cloudflare 的网络，这其中就包括果园编程节省的碳排放。

我们的绿色技术实践

图像优化

使用 Cloudflare 的图像重设大小功能调整站点上的视觉内容。适当调整图像大小可以减少每个最终用户加载页面所需的能量，从而减少下游的碳排放量。

云服务整合

使用 Cloudflare 的云安全和性能服务，如 WAF、网络防火墙和 DDoS 缓解。使用的云服务越多，可以关掉的本地设备就越多。停用硬件将减少碳排放量！
