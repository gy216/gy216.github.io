Scratch打包器（果园精简版）

Standalone v3.6.0 (2025-06-12) - https://packager.turbowarp.org/

将 Scratch 作品转换为 HTML 文件、ZIP 压缩包或 Windows、MacOS 和 Linux 的可执行文件。

如果您只是想以一种简单的方式将 Scratch 官网作品嵌入您的网站，您可能会对 网站嵌入 感兴趣。

在 果园官网 或者 果园的GitHub 报告错误，提供反馈，并提出建议。

此网站不属于 Scratch、Scratch 团队或 Scratch 基金会。

选择作品

您可以通过复制粘贴 Scratch 作品链接，或使用其他模式来选择要打包的作品。 如果您打包的作品并非原创，请做好署名。

未分享的作品将不会显示。

您可以尝试分享作品或者将作品保存到设备（文件 > 保存到电脑）并在打包机中打开。打包程序生成的文件是完全独立的，不受 Scratch API 更新的影响。

有关更多信息和解决方法，请访问TW的官网：https://docs.turbowarp.org/unshared-projects
