服务介绍

NTP（Network Time Protocol）是一种旨在将网络上计算机时钟同步到统一时间基准的协议。NTP确保了全球数十亿设备、海底甚至太空设备的可靠性。

准确的计时对于许多已经革新且对我们日常生活至关重要的应用至关重要：

卫星通信 GPS定位 5G网络 金融服务 医疗健康

"果园编程NTP项目致力于在NTP协议中进行研究与开发，为全球时间同步提供可靠服务。"

Windows 7/8/8.1/10 客户端配置

1 打开"控制面板" > "时钟、语言和区域" > "日期和时间"
2 点击"Internet时间"选项卡 > "更改设置"
3 勾选"与 Internet 时间服务器同步"
4 在"服务器"一栏填入：
time.gybc.top
或
time1.gybc.top
5 点击"立即更新" > "确定"保存设置

macOS 配置

1 点击屏幕左上角苹果菜单 > "系统偏好设置"
2 选择"日期与时间"
3 点击锁图标并输入密码以解锁设置
4 在"日期与时间"选项卡中，取消勾选"自动设置日期与时间"
5 点击"时区"选项卡，确保时区设置正确
6 返回"日期与时间"选项卡，点击"高级"按钮
7 在NTP服务器字段中填入：
time.gybc.top
或
time1.gybc.top
8 勾选"使用网络时间"并设置同步频率

Linux 配置

Linux系统配置NTP客户端有多种方法，具体取决于您使用的发行版和初始化系统。

常用配置方法：

1. 使用timedatectl（Systemd系统）
sudo timedatectl set-ntp true
sudo timedatectl set-ntp false
sudo timedatectl set-timezone Asia/Shanghai
2. 编辑ntp.conf文件
server time.gybc.top iburst
server time1.gybc.top iburst
3. 使用chrony（现代推荐）
server time.gybc.top iburst
server time1.gybc.top iburst

提示：由于Linux系统配置方法多样，建议根据您的具体发行版搜索详细配置教程。能使用Linux的用户通常具备一定的技术能力。
